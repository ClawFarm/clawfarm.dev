# clawfarm.dev

Landing page for [ClawFarm](https://github.com/clawfarm/clawfarm). Built with Astro + Tailwind, deployed to GitHub Pages.

## Development

```bash
npm install
npm run dev       # http://localhost:4321
npm run build     # Build to ./dist/
```

## Deployment

Pushes to `master` automatically deploy via GitHub Actions. Custom domain configured in GitHub Pages settings.
